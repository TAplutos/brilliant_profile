Inscribing Regular Polygons in Circles

Geometry Level 5

Let n≥3n \ge 3n≥3. A nnn-sided regular polygon is inscribed in a circle of radius 111. Then, all the diagonals of the polygon are drawn as well. Thus every vertex of the polygon is connected by a line segment to every other vertex.

Let LnL_nLn​ be the total length of line segments that are drawn by this process (including the perimeter). To the nearest thousandth, find the value oflim⁡n→∞Lnn2\displaystyle\lim_{n\to\infty}\frac{L_n}{n^2}n→∞lim​n2Ln​​.


The answer is 0.637.

This is a nice solution! I think it is valid if you explain it properly and rewrite it a bit. The average length is simply the total length divided by the number of line segments, so therefore multiplying the average by the number of line segments gives you the total length. And as n→∞n \to \inftyn→∞, the average approaches the distribution function with the value of 4π\dfrac{4}{\pi}π4​, while the number of line segments divided by n2n^2n2 approaches 12\dfrac{1}{2}21​. In general the limit of a product is the product of the limits so it is okay to calculate the limits separately and then multiply them.

One correction though: in the question it asks for the total value of all line segments drawn, so it's not just the diagonals but the side lengths of the polygon as well. So instead of n(n−3)2\dfrac{n(n-3)}{2}2n(n−3)​, you should have written n(n−1)2\dfrac{n(n-1)}{2}2n(n−1)​. Fortunately this does not change the limit and so you got the same answer; but I thought it's worth mentioning.
