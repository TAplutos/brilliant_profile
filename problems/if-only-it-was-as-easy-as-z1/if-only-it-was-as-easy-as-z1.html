If only it were as easy as 1, 2, 3

Algebra Level 3

If zzz is a complex number satisfying the equation

z+1z=z2+1z2<0,z+\frac{1}{z}=z^2+\frac{1}{z^2}<0,z+z1​=z2+z21​<0,

what is the value of z3+1z3z^3+\frac{1}{z^3}z3+z31​?


The answer is 2.

Before I begin my solution, let me explain what's so special about z+1zz+\frac{1}{z}z+z1​.

First, consider the equation

z+1z=2cos⁡θz+\frac{1}{z}=2\cos \thetaz+z1​=2cosθ

Solve for zzz, we get

z=cos⁡θ+isin⁡θz=\cos\theta + i\sin \thetaz=cosθ+isinθ

Looks familiar? Yes, it is the well-known DeMoivre's Theorem!

Hence, for any integer nnn, we have

zn+1zn=2cos⁡nθz^n+\frac{1}{z^n}=2\cos {n\theta}zn+zn1​=2cosnθ

Using the method stated above,

z+1z=z2+1z2z+\frac{1}{z}=z^2+\frac{1}{z^2}z+z1​=z2+z21​

2cos⁡θ=2cos⁡2θ2\cos \theta = 2\cos{2\theta}2cosθ=2cos2θ

Solve the equation we will get

θ=120∘\theta=120^\circθ=120∘

*Note that the solution θ=0∘\theta=0^\circθ=0∘ is not suitable in this case.

Substitute to the expression

z3+1z3=2cos⁡3(120∘)z^3+\frac{1}{z^3}=2\cos{3(120^\circ)}z3+z31​=2cos3(120∘)

We get the desired answer 222.
