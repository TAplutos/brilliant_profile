Silhouette of a Fish

Calculus Level 3

Find the minimum possible value of ∣a∣|a|∣a∣ such that the function below is continuous along the interval [b,∞)[b,\infty)[b,∞) and there is no value of x<bx<bx<b such that yyy is a real number.

16y2=a+15x−9x2+x316y^2=a+15 x-9 x^2+x^316y2=a+15x−9x2+x3

Note: bbb is a constant of real integral value.
