Mistakes Give Rise to Problems

Algebra Level 3

We know am+n=am×ana^{m+n}=a^m \times a^nam+n=am×an But there are some students who claim that am+n=am+ana^{m+n}=a^m+a^nam+n=am+an which is obviously incorrect. However, it is true for some triplets of integers, (a,m,n)(a,m,n)(a,m,n). If a∈[0,10]a\in[0,10]a∈[0,10] and m,n∈[1,10]m,n\in[1,10]m,n∈[1,10], then find the number of possible triplets for which am+n=am+ana^{m+n}=a^m+a^nam+n=am+an.


The answer is 101.

First, the case when a=0 works for all cases because 0^x=0 and 0+0=0. Also, since there are 10 possible values for m,n, there are 100 possible triples of (0,m,n).

Next for a>0, we divide both sides by a^n and get am=am−n+1a^m=a^{m-n}+1am=am−n+1. This means that the difference between ama^mam and am−na^{m-n}am−n must be 1. However, this is only achievable if a=2 and m=n=1. The proof that this doesn't work is that there is no positive integer that when raised to a power is 1 greater than itself except for 2. Finally, we add the 100 cases when a=0 and the special case when a=2 and our total count is 101.

Side note: if an integer x>2 and integers p>n>0, xp≠xp−n+1x^p\neq x^{p-n}+1xp​=xp−n+1
