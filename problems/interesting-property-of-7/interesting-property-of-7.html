Interesting Property Of Seven!

Number Theory Level 4

Note that 7!=10×9×8×7.7!=10 \times9 \times8\times7.7!=10×9×8×7.

What is the largest possible value of nnn for which n!n!n! can be represented as a product of n−3n-3n−3 consecutive integers?


The answer is 23.

My first approach was to do some brute force hand calcs to get a feel for the question and try to pick up any patterns. After a while I realised n!3!<n!\frac{n!}{3!} < n!3!n!​<n! and therefore n!3!<(n+k)!(3+k)!<n!\frac{n!}{3!} < \frac{(n+k)!}{(3+k)!} < n!3!n!​<(3+k)!(n+k)!​<n! for certain values of k up to some limit.

So if we want to find a solution we can set the second two parts of the inequality to be equal: (n+k)!(3+k)!=n!\frac{(n+k)!}{(3+k)!} = n!(3+k)!(n+k)!​=n! I figured since both sides have n! in some forms I could probably cancel it out... And with some more algebra I arrived at this product series ∏i=1k(n+i)=3!⋅∏i=1k(3+k)\prod_{i=1}^k (n+i) = 3!\cdot \prod_{i=1}^k (3+k)i=1∏k​(n+i)=3!⋅i=1∏k​(3+k)

And here the RHS does not change as you increase n, and I also thought (since the question suggested there was a maximum number) k would be 1 for the largest n, and so we have n+1=3!∗(3+1)n+1=3!*(3+1)n+1=3!∗(3+1) and so n=23n=\boxed{23}n=23​

I think I was half trying to find a simple iterative equation I could plug into python but it was much more satisfying arriving at the solution on paper. Great question!
