This solution doesn't explain how to calculate the area, but instead focuses on why we tend to get circles.

Here's the best explanation that I can come up with

1) Since csc⁡(x+2π)=csc⁡x \csc ( x + 2\pi ) = \csc x csc(x+2π)=cscx, it suffices to look at the the region that 0≤x≤2π 0 \leq x \leq 2 \pi 0≤x≤2π. Since csc⁡xcsc⁡y>0 \csc x \csc y > 0 cscxcscy>0, we will look at 0≤x,y≤π 0 \leq x , y \leq \pi 0≤x,y≤π.

2) We want to understand why the circles seem to form about the points (π2,π2) (\frac{ \pi}{2 }, \frac{ \pi }{2} ) (2π​,2π​). To do so, we will simply take the Taylor expansion about that point. This gives us:

(1+12(x−π2)2+524(x−π2)4+…)(1+12(y−π2)2+524(y−π2)4+…)=ϕ ( 1 + \frac{1}{2} ( x - \frac{\pi}{2} ) ^2 + \frac{5}{24} ( x - \frac{ \pi }{2} ) ^ 4 + \ldots ) ( 1 + \frac{1}{2} ( y - \frac{\pi}{2} ) ^2 + \frac{5}{24} ( y - \frac{ \pi }{2} ) ^ 4 + \ldots ) = \phi (1+21​(x−2π​)2+245​(x−2π​)4+…)(1+21​(y−2π​)2+245​(y−2π​)4+…)=ϕ

Observe that the RHS only has positive coefficients and even exponents, hence is an increasing function on x≥π2 x \geq \frac { \pi } {2} x≥2π​. (It is also an "increasing function" on x≤π2 x \leq \frac{\pi}{2} x≤2π​ if we trace our steps backwards, but that doesn't really matter ).

3) For small values of a,b a, b a,b, and k≳1 k \gtrsim 1 k≳1 the solution to

(1+a2)(1+b2)=k ( 1 + a^2 ) ( 1 + b^2 ) = k (1+a2)(1+b2)=k

looks very much like the circle a2+b2=(k−1) a^2 + b^2 = (k-1) a2+b2=(k−1). This is obvious, because we are dropping the "very small term of a2b2 a^2 b^2 a2b2".

Thus, this explains why with ϕ \phi ϕ, we obtain the circles.
