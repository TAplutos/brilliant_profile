Now there's an nnn?

Calculus Level 4

Solve for the maximum positive integral value of nnn such that there is exactly 1 positive rational solution for aaa.

Note first that the sum of a series of logs (of the same base) is equal to the log of the product of those terms. Now we can write this product as

a(a+2)(a−2)(a+4)∗(a+1)(a+3)(a−1)(a+5)∗(a+2)(a+4)a(a+6)∗(a+3)(a+5)(a+1)(a+7)∗.....\dfrac{a(a + 2)}{(a - 2)(a + 4)}*\dfrac{(a + 1)(a + 3)}{(a - 1)(a + 5)}*\dfrac{(a + 2)(a + 4)}{a(a + 6)}*\dfrac{(a + 3)(a + 5)}{(a + 1)(a + 7)}*.....(a−2)(a+4)a(a+2)​∗(a−1)(a+5)(a+1)(a+3)​∗a(a+6)(a+2)(a+4)​∗(a+1)(a+7)(a+3)(a+5)​∗.....

This is a telescoping product that leaves us with the equation

log⁡n((a+2)(a+3)(a−2)(a−1))=1⟹(a+2)(a+3)=n(a−2)(a−1)\log_{n}\left(\dfrac{(a + 2)(a + 3)}{(a - 2)(a - 1)}\right) = 1 \Longrightarrow (a + 2)(a + 3) = n(a - 2)(a - 1)logn​((a−2)(a−1)(a+2)(a+3)​)=1⟹(a+2)(a+3)=n(a−2)(a−1)

⟹(n−1)a2−(3n+5)a+2(n−3)=0\Longrightarrow (n - 1)a^{2} - (3n + 5)a + 2(n - 3) = 0⟹(n−1)a2−(3n+5)a+2(n−3)=0

⟹a=(3n+5)±(3n+5)2−8(n−1)(n−3)2(n−1)\Longrightarrow a = \dfrac{(3n + 5) \pm \sqrt{(3n + 5)^{2} - 8(n - 1)(n - 3)}}{2(n - 1)}⟹a=2(n−1)(3n+5)±(3n+5)2−8(n−1)(n−3)​​

⟹a=(3n+5)±n2+62n+12(n−1).\Longrightarrow a = \dfrac{(3n + 5) \pm \sqrt{n^{2} + 62n + 1}}{2(n - 1)}.⟹a=2(n−1)(3n+5)±n2+62n+1​​.

Now in order for aaa to be rational with nnn being integral we will require that the polynomial under the square root sign be a perfect square, i.e.,

n2+62n+1=m2⟹(n+31)2−960=m2n^{2} + 62n + 1 = m^{2} \Longrightarrow (n + 31)^{2} - 960 = m^{2}n2+62n+1=m2⟹(n+31)2−960=m2

⟹[(n+31)+m][(n+31)−m]=960=26×3×5.\Longrightarrow [(n + 31) + m][(n + 31) - m] = 960 = 2^{6} \times 3 \times 5.⟹[(n+31)+m][(n+31)−m]=960=26×3×5.

Now if we choose divisor pairs aaa and bbb we will end up with the pair of equations

(n+31)+m=a,(n+31)−m=b⟹n=a+b2−31.(n + 31) + m = a, (n + 31) - m = b \Longrightarrow n = \dfrac{a + b}{2} - 31.(n+31)+m=a,(n+31)−m=b⟹n=2a+b​−31.

This value is integral and maximized when a+ba + ba+b is even and when aaa and bbb are as "far apart" as possible. The most "far apart" divisor pair is (960,1),(960,1),(960,1), but the sum here is odd. The next pair to consider is (480,2),(480,2),(480,2), which yields a potential maximal value of n=210.n = 210.n=210. But first we need to check which values of aaa result:

a=635±239438⟹a=2311a = \dfrac{635 \pm 239}{438} \Longrightarrow a = \dfrac{23}{11}a=438635±239​⟹a=1123​ or a=1819.a = \dfrac{18}{19}.a=1918​.

Now looking back at the original sum, we see that the domain conditions of the log function require that a>2.a \gt 2.a>2. Thus with this value of nnn we have exactly one valid rational solution for a,a,a, namely a=2311,a = \dfrac{23}{11},a=1123​, and so the desired value for nnn is 210.\boxed{210}.210​.

Thanks! Solution and root are essentially interchangeable, but the phrasing is a bit different. For example, the "solutions" of the equation ax2+bc+c=0ax^{2} + bc + c = 0ax2+bc+c=0 are the same as the "roots" of the polynomial ax2+bx+c.ax^{2} + bx + c.ax2+bx+c. In other words, when we talk about the roots of an expression it is implied that we equating that expression to 0.0.0.
