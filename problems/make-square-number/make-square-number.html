Make square number

Number Theory Level 3

Find the sum of all integers n n n which make 28+211+2n 2^{8}+2^{11}+2^{n} 28+211+2n a perfect square.

(1981 HMO modification)


The answer is 12.

Note first that 28+211=2304=4822^{8} + 2^{11} = 2304 = 48^{2}28+211=2304=482.

So we are looking for integers nnn such that

482+2n=m2⟹2n=m2−482=(m−48)(m+48)48^{2} + 2^{n} = m^{2} \Longrightarrow 2^{n} = m^{2} - 48^{2} = (m - 48)(m + 48)482+2n=m2⟹2n=m2−482=(m−48)(m+48)

for some integer mmm.

By the Fundamental Theorem of Arithmetic we must have both of ∣m−48∣|m - 48|∣m−48∣ and ∣m+48∣|m + 48|∣m+48∣ be powers of 222. The only such powers that differ by 2∗48=962 * 48 = 962∗48=96 are 25=322^{5} = 3225=32 and 27=1282^{7} = 12827=128. As a result, we must have 2n=25∗27=2122^{n} = 2^{5} * 2^{7} = 2^{12}2n=25∗27=212, thus giving us 12\boxed{12}12​ as the solution.

(Note there are two possible values for mmm, namely 808080 and −80-80−80, but both of course lead to the same value of nnn.)

Because this is a perfect square, it must have the representation (2a+2b)2(2^a+2^b)^2(2a+2b)2 where a,b are integers.

Upon squaring, we get 22a+2a+b+1+22b2^{2a}+2^{a+b+1}+2^{2b}22a+2a+b+1+22b

We can observe that out of the three terms, only one can have an odd power, thus n must be even.

In this case, we let a=4 so our first term is 282^828 and our second term is 25+b⇒25+b=211 ∴b=6.2^{5+b}\Rightarrow 2^{5+b}=2^{11} ~ \therefore b=6.25+b⇒25+b=211 ∴b=6. since n=2b, n=12\boxed{n=12}n=12​
