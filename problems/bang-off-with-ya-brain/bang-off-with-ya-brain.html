View each pair (ai,bi)(a_i,b_i)(ai​,bi​) as a ordered pair in the Cartesian plane.

Now each square root ai2+bi2\sqrt{a_i^2+b_i^2}ai2​+bi2​​ represents the distance from the origin to that point (the hypotenuse).

Assume that we put each hypotenuse tip to tail so that they extend in a line. This "trail of hypotenuses" no matter what each individual length is, will end at the point (p,q)(p,q)(p,q)

Thus the overall length this trial will represent our sum. Which is obviously minimized when this length is straight. Or aibi=λ\dfrac{a_i}{b_i}=\lambdabi​ai​​=λ
