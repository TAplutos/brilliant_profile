Minimum value?

Algebra Level 2

Given that x>−1x>-1x>−1, find the minimum value of

4x2+8x+136(x+1)\dfrac{4x^{2}+8x+13}{6(x+1)}6(x+1)4x2+8x+13​

There's a easier way!


The answer is 2.

Substitute a=x+1a=x+1a=x+1

We are left with

4a2+96a\dfrac{4a^2+9}{6a}6a4a2+9​

4a26a+96a\dfrac{4a^2}{6a}+\dfrac{9}{6a}6a4a2​+6a9​

2a3+32a\dfrac{2a}{3}+\dfrac{3}{2a}32a​+2a3​

Substitute p=2a3p=\dfrac{2a}{3}p=32a​

p+1pp+\frac{1}{p}p+p1​

This is obviously maximized at p=1 (or you could use am-gm).

Therefore, p=2a3=1p=\dfrac{2a}{3}=1p=32a​=1

a=32a=\frac{3}{2}a=23​

x+1=32x+1=\frac{3}{2}x+1=23​

x=12x=\frac{1}{2}x=21​

Plugging x back in, we find the original function to equal 2.
