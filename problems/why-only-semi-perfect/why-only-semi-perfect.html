Why Only Semi-Perfect

Number Theory Level 5

A multiplicative perfect number is a positive integer nnn who's divisors' product totals to n2n^2n2.

For example, the factors of 222222 are 1,2,11,22,  1⋅2⋅11⋅22=4841,2,11,22,~~1\cdot2\cdot11\cdot22=4841,2,11,22,  1⋅2⋅11⋅22=484

Define the term semi-multiplicative perfect number as a positive integer who's divisors' product (including the number itself) totals to n3n^3n3. Find the sum of the first 8 semi-multiplicative perfect numbers.


The answer is 200.

public static void main(String args[]){
    int sum = 0;
    for(int i = 1, n = 0; n != 8; i++)
        if(factorProduct(i) == i * i * i){
            sum += i; n++;
        }

}

private static int factorProduct(int n){
    int prod = n;
    for(int i = 2; i <= n / 2; i++)
        if(n % i == 0) prod *= i;
    return prod;
}
