[Debate] The Infamous Magic of Zero

Debate in the comments if 000^000 should be 111, 000 or undef

000^000 is defined to be 1 by the IEEE.

"Indeterminate" means that there exists more than one alternative evaluation of the expression, which is clearly the case here. Hence, it's indeterminate. Given any value xxx, some argument can be made that 00=x{ 0 }^{ 0 }=x00=x. So, people can choose one value of xxx as a "convention", but that's all it is, a convention, and not a derived mathematical fact.

Mathematics does not say that every expression necessarily evaluates to an unique value.
